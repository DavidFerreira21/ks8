# FreeLens

Interface grafica gratuita e open source para gerenciar clusters Kubernetes. E um aplicativo standalone compativel com macOS, Windows e Linux, e tem como objetivo simplificar o gerenciamento do Kubernetes com uma UI intuitiva e facil de usar.

https://github.com/freelensapp/freelens
